# To-Do List App

A simple and user-friendly To-Do List application built to manage daily tasks efficiently.
Users can add new tasks, mark tasks as completed, and delete tasks when finished. 
The project provides a clean and responsive interface to improve productivity and task organization.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and responsive UI
- Easy task management

## Technologies Used
- HTML
- CSS
- JavaScript
